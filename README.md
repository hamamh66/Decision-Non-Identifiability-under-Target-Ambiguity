# Decision Non-Identifiability under Target Ambiguity

## A Falsifiability Certificate for Composite-Target Machine Learning

Official implementation accompanying the paper **"Decision
Non-Identifiability under Target Ambiguity: A Falsifiability Certificate
for Composite-Target Machine Learning."**

This repository provides:

-   Implementation of the Decision Identifiability Framework.
-   Computation of the Certified Decision Fraction (κ).
-   Construction of defensibility sets for composite targets.
-   Reproducible experiments across multiple datasets and
    machine-learning models.
-   Scripts to reproduce all figures, tables, and results reported in
    the paper.

More documentation will be added with future releases.
